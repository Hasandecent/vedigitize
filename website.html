Cms Based Website

If you need to update your website frequently, Adding images and content regularly, You should consider a CMS based website instead of a static website. A CMS ( Content management system) Based website allows you to add and edit content, images ect from an admin panel which is secured with a password. These systems are useful because you don’t need to code in HTML you just need to login to administrative panel and easily add edit content from a wysiwyg editor.
